Sobre o ChessClub

É uma aplicação desenvolvida durante o Projeto de Extensão Xadrez Lógico.

Objetivo: O sistema visa organizar o funcionamento dos clubes de xadrez escolares na cidade de Assaré-CE.


Inclui funcionalidades como:

  • Cadastro de clubes
  • Cadastro de jogadores
  • Cálculos de rating entre jogadores
  • Rankeamentos
  • Sessão de notícias

Desenvolvido utilizando Flask, Python, e outras tecnologias, este projeto foi criado com o intuito de promover a aprendizagem do discente.
